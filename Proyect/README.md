# Proyecto-D&D

Este proyecto consiste en aprender a utilizar GitHub,
crear ramas, trabajar en los commits y comprobar que las ramas que estamos creando funcionan y luego podemos unirlas al main.

## Estructura del proyecto

El proyecto está compuesto por las siguientes clases:

- `Personaje`: Clase base con atributos comunes como nombre y vida.
- `Guerrero`: Clase que hereda de Personaje y añade habilidades de ataque físico.
- `Mago`: Clase que hereda de Personaje y permite lanzar hechizos.
- `App`: Clase principal desde donde se ejecuta el programa.
